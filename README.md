# Discord Pomodoro Bot

A Discord bot that uses the Pomodoro technique to help users focus during voice calls.
The bot automatically mutes and unmutes users in a voice channel based on work and break intervals.

This project is still under development. Future plans include packaging the bot as an executable file.

## Features
- Voice-based Pomodoro sessions
- Automatic mute/unmute during focus and break phases
- Auto-mute users who join during a focus session
- Simple text commands

## Commands 
!join  
!pomodoro <work_minutes> <break_minutes>  
!stop  
!leave
